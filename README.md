# Text_Editor
![Text_Editor](https://user-images.githubusercontent.com/96577772/185872707-a9bb546d-43b0-4a81-b5b8-e1bd2d01584d.jpg)<br/>
 --> A java based text editor with stack type clipboard to keep multiple strings in clipboard <br/>
 --> Graphical UI made using Java Swing<br/>
 --> A menu bar with file handling (open, new, save, print) and editing options (cut, copy, paste)<br/>
 --> A stack-based copy paste to store more than 1 string on clipboard<br/>
 --> Formating options to change font type, font size, font style for the whole file<br/>
